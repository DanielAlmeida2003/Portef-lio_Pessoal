    ReManage

    Desenvolvi uma aplicação que permite gerir Candidatos, Clientes e as oportunidades para os candidatos, expedir propostas para candidatos e para clientes.

    Podem também ser inseridas reuniões com candidatos, clientes ou utilizadores, com funcionalidades diferentes caso o cargo do utilizador seja mais elevado.

    Na aplicação, podemos realizar testes aos candidatos, neste caso será as qualificações, onde podem participar clientes, utilizadores, candidatos.

    Finalmente, existem funcionalidades adicionais para administrar a aplicação, como por exemplo: CRS (category-roles-skills), Cidades, Sources e outros mais.

    Esta aplicação, demos o nome de ReManage.
